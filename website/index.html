FAQ

Frequently Asked Questions

Everything you need to know before using Underseerr.

Do I need an Overseerr server?

Yes. Underseerr is a third-party client application. You must have an existing Overseerr instance running (e.g., on your home server or NAS) to use this app. You will need your server URL and API key/credentials to log in.

Can I stream movies with this app?

No. Underseerr is strictly a request management tool. It does not stream, download, or host any media content. It simply allows you to request content to be added to your media library managed by Overseerr.

Is Underseerr free?

Underseerr is open-source and free to use for all core features. We offer an optional Underseerr Premium subscription that unlocks specific conveniences like our hosted push notification relay (so you don't need to self-host one) and exclusive cosmetic themes ("Vibrant"). The app remains ad-free and fully functional without a subscription.

Is there an iOS version?

Currently, Underseerr is available primarily for Android. Since it is built with Kotlin Multiplatform, an iOS version is technically possible in the future, but our primary focus right now is delivering the best Android experience.
