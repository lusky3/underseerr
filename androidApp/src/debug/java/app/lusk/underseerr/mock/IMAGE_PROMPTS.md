## Usage Instructions

1. Use these prompts with AI image generators (Midjourney, DALL-E, Stable Diffusion, etc.)
2. Generate at recommended resolutions for your target display
3. Save as JPG files with naming convention: `movie_<ID>.jpg` or `tv_<ID>.jpg`
4. Place in appropriate folders:
   - Posters: `androidApp/src/debug/assets/mock_images/posters/`
   - Backdrops: `androidApp/src/debug/assets/mock_images/backdrops/`

## Tips for Best Results

- Always specify "no text" to avoid watermarks or unwanted labels
- Use vertical format (2:3) for posters to match typical movie poster dimensions
- Use horizontal format (16:9) for backdrops/headers
- Adjust prompts as needed for your specific AI tool
- Generate at higher resolution then downscale for best quality
